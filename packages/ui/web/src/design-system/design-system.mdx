## **Spacing**

### Overview

To control padding and margin. TailwindCSS padding and margin classes, no import is required.

### Usage

**To add padding (p) or margin (m):**

```
p-{number}
m-{number}
```

**To control the padding or margin on one side:**

- t - top of the element
- r - right side of the element
- b - bottom of the element
- l - left side of the element

```
p{t|r|b|l}-{number}
m{t|r|b|l}-{number}
```

**To control the horizontal or vertical padding or margin:**

- x - horizontal
- y - vertical

```
p{x|y}-{number}
m{x|y}-{number}
```

**Example:**

```html
<div className="p-10">Add a padding of 10 to all sides</div>
```

```html
<div className="mx-4">Add a horizontal margin of 4</div>
```

## **Colors**

### Overview

Colors defined in tailwind.config.js, no import is required.

Includes:

- Primary: _For buttons, headers etc._
- Secondary
- Black: _For headings and body text, primarly headings_
- Gray: _For headings and body text, primarly body text_
- White: _For backgrounds_
- Warning: _For warnings_
- Error: _For errors_

### Usage

**Background color:**

```
bg-{color}

```

**Text color:**

```
text-{color}

```

**Border color:**

```
border-{color}

```

**Example:**

```html
<div className="bg-secondary-default text-white">
  Background with gray color (secondary default) and white text
</div>
```

## **Buttons**

### Overview

Buttons can take different properties.

- primary, _boolean_
- size - _'small' | 'medium' | 'large'_
- additionalClassName - _string_, optional

By default, **primary=false** and **size='medium'**.

### Usage

```javascript
import { Button } from '../ui/common/Button'

<Button primary size="medium"}>
  Button
</Button>
```
